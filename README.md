# YGGDRASH CLI
YGGDRASH CLI allows you to communicate with a remote or local node

## Requirements
- Node.js
- NPM
- [YGGDRASH Node](https://github.com/yggdrash/yggdrash)

## Installation
```sh
$ npm install -g @yggdrash/cli
```

## Usage

```
$ ygg <command> [options]
```

### Example
Send transaction
```
$ ygg tx send --from ace --to bob --value 10
```
